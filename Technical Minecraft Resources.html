Example Redstone Code (Function Command)

Example of a repeating function to control a piston:

# functions/piston_control.mcfunction
execute if block 0 64 0 minecraft:redstone_block run setblock 0 65 0 minecraft:piston[facing=up]
execute unless block 0 64 0 minecraft:redstone_block run setblock 0 65 0 minecraft:air

Example Farm Calculation (Rates)

Example calculation of a sugarcane farm's theoretical output. Assume a row of 10 sugarcane, growing every 18 seconds on average:

$$ \text{Rate} = \frac{10 \text{ sugarcane}}{18 \text{ seconds}} \approx 0.55 \text{ sugarcane/second} $$
